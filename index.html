当前的问题

你的维格表分享链接目前是 「仅查看」 模式。成员想通过程序批量新增数据，需要 API 访问权限。

💡

一句话解释

API Token 就像一把 数字钥匙，有了它，程序就能代替人工在表格里新增、修改数据。

Token 的权限 = 用户在界面上的操作权限

📋 三种方案，任选其一

⚡

方案 A

最快

分享你的 API Token 给成员，程序直接操作表格。

⏱ 耗时 2 分钟
🛡️

方案 B

正规做法

把成员加入空间站，他自己生成 Token，权限精准可控。

⏱ 耗时 5 分钟
📝

方案 C

仅手动

重新分享「可编辑」链接，但只能浏览器手动操作，不支持 API。

⏱ 耗时 1 分钟

⚡ 方案 A：分享你的 API Token

1

登录维格云

打开浏览器访问 vika.cn，登录你的账号。

2

进入「开发者配置」

点击左下角个人头像 → 用户中心 → 开发者配置

👤 头像 → ⚙️ 用户中心 → 🔧 开发者配置
3

生成 API Token

点击 "+" 按钮生成新 Token。首次使用需绑定邮箱。

4

复制并发送

Token 格式类似 uskYtInkHozfsMikhh0yfoS — 复制后发给成员即可。

🔒 用完后的安全操作

5

重置 Token（重要！）

成员用完 API 后，回到「开发者配置」页面，点击 「重新生成」，旧 Token 立即失效。

⚠️

安全提醒：Token 相当于你的账号密码，只发给你信任的人。用完记得重置！

🛡️ 方案 B：邀请成员加入空间站

1

进入空间站设置

左侧边栏点击空间站名称旁的 ⚙️ 设置图标。

2

管理成员

进入「成员管理」。如人数已满，可移除不活跃成员或升级套餐。

3

发送邀请

点击「邀请成员」→ 复制链接发给成员。

4

成员生成 Token

成员登录后，按方案 A 步骤 1-4 自己生成 API Token。
